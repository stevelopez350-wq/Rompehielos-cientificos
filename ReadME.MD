# 🎴 Rompehielos Científicos · Aula IA

Aplicación web interactiva diseñada para dinamizar las clases de **Ciencias Naturales** 
mediante 7 barajas temáticas de cartas digitales. Cada baraja contiene aproximadamente 
30 actividades breves (≤ 5 minutos) que estimulan distintas habilidades del pensamiento 
científico.

## 🎯 Barajas incluidas

| Símbolo | Baraja | Habilidad que desarrolla |
|---------|--------|--------------------------|
| α (Alpha)   | Observación      | Atención al detalle y descripción |
| Σ (Sigma)   | Hipótesis        | Formulación de predicciones |
| Δ (Delta)   | ¿Qué pasaría si? | Pensamiento crítico y especulativo |
| Ψ (Psi)     | Método Científico| Diseño experimental |
| Λ (Lambda)  | Conexiones       | Relación entre conceptos |
| Ω (Omega)   | Misterio         | Resolución de enigmas |
| Φ (Phi)     | Role‑Play        | Empatía científica y comunicación |

## ✨ Características

- 🎲 **Selector aleatorio** de estudiantes (1‑26) con modos individual, pareja y grupo.
- 🃏 **Mazo visual estilo Hearthstone**: las cartas vuelan desde la pila, se voltean y 
  se depositan en una pila de descarte.
- 🎨 **Diseño temático único** por baraja con fuentes, gradientes y animaciones propias.
- 📱 **Responsive**: funciona en proyector, laptop, tablet o móvil.
- ⚡ **Sin instalación**: 100 % web, abre el enlace y listo.

## 🚀 Cómo usarla

Visita: **https://[tu-usuario].github.io/rompehielos-cientificos/**

1. Elige una baraja del menú principal.
2. Haz clic sobre el mazo para sacar una carta.
3. Toca la carta para revelar la actividad.
4. Usa la ruleta para seleccionar al estudiante que participará.
5. ¡A jugar y aprender!
